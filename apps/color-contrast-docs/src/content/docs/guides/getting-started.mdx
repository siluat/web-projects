---
title: Getting Started
description: Install and start using @siluat/color-contrast for WCAG contrast ratio computation.
---

## Installation

```bash
npm install @siluat/color-contrast
```

## Quick Example

```typescript
import { contrastRatio, checkContrast } from '@siluat/color-contrast';

const ratio = contrastRatio('#ffffff', '#000000');
// 21

const result = checkContrast('#333333', '#ffffff');
// { ratio: 12.63, normalText: 'AAA', largeText: 'AAA' }
```

## CLI

A command-line interface is also available:

```bash
npm install -g @siluat/color-contrast-cli
```

```bash
ccr "#ffffff" "#000000"
```
