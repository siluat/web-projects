## Installation

Run directly with npx (no install required):

```bash
npx @siluat/color-contrast-cli ccr '#ffffff' '#000000'
```

Or install globally:

```bash
npm install -g @siluat/color-contrast-cli
```

```bash
ccr '#ffffff' '#000000'
```

## Quick Example

```bash
ccr '#333333' '#ffffff'
# Ratio    12.63:1
# Normal   AAA ✅
# Large    AAA ✅
```

Use `--suggest` to get an accessible alternative when contrast fails:

```bash
ccr '#999999' '#ffffff' --suggest
```

Use `--batch` to audit multiple pairs from a file:

```bash
ccr --batch < palette.txt
```

## Library

The computation engine is available as a standalone JavaScript/TypeScript library:

```bash
npm install @siluat/color-contrast
```

```typescript
import { contrastRatio, checkContrast } from '@siluat/color-contrast';

const ratio = contrastRatio('#ffffff', '#000000');
// 21

const result = checkContrast('#333333', '#ffffff');
// { ratio: 12.63, normalText: 'AAA', largeText: 'AAA' }
```

See the [Library API reference](/docs/reference/library-api) for full documentation.
